# testpkg

Small demo package for testing pip packaging workflows.

Install locally:

    python -m pip install --upgrade build
    python -m build
    python -m pip install dist/safe-chain-pi-test-0.0.1-py3-none-any.whl

Run tests:

    python -m pip install pytest
    pytest -q
